Re-imagining|

Re-imagining has been a common theme in my journey through this class. Re-imagining gender, re-imagining words - as stated before, things are often more complicated than we think they are.

I think that this project falls into that category -

Like not centering this text

Or changing up the size

Or the color

Maybe I'm not trying to sell you something. Maybe I'm not trying to make it accesible and optimized.

Doing something different is always hard. Conforming has always been easier, even if what we try to conform to is impossible.

As Tolentino writes:

"Figuring out how to 'get better' at being a woman is a ridiculous and often amoral project. Satisfaction remains, under the terms of the system, necessarily out of reach."

And that's the thing about code - it's never perfect. No matter how hard you edit and type and code, there will always be more work to be done.

In that sense, I think that Feminism and code have a lot of overlap. There is always more that can be done, refinments and improvements in the system, be that a code editor or our society.
